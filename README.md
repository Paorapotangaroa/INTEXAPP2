This was built as part of a school project. We were tasked with creating a new website for the BYU Egypt project. We recieved the requirements in a meeting that ended at 10:00 AM on Monday. Presentations and final products were due on Friday at 12:00 PM.

## Requirements: 
- Normalize the DB (PostgreSQL originally but we migrated to MySQL)
- CRUD functionality using the ASP.NET framework
- Build & Integrate a Supervised Machine Learning Model to predict resting position of future mummies
- Develop an unsupervised ML model and present new findings to researchers
- Secured using MFA & 3rd Party Login
- Hosted on AWS with TLS/HTTPS
